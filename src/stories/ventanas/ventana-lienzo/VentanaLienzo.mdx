# VentanaLienzo
El componente `VentanaLienzo` es un contenedor en blanco. Incluye el menú lateral y una barra superior para el breadcrumb.

## Ejemplo de uso

- MiComponente.tsx
```tsx
import { VentanaLienzo } from 'react-ecosistema-unp/shared';

const MiComponente: React.FC = () => {
  return (
    <VentanaLienzo>
    <div>
      Contenido...
    </div>
  </VentanaLienzo>
  );
}
```
