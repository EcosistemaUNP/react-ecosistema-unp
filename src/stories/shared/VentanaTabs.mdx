# VentanaTabs
El componente `VentanaTabs` es un contenedor en blanco con tabs en la parte superior. Este incluye el menú lateral del ecosistema.

## Ejemplo de uso

> **Nota:** El componente debe estar envuelto entre el componente `AuthProvider` de **react-ecosistema-unp/utils** y el componente `BrowserRouter` de **react-dom**, es de preferencia hacerlo en el componente *index.tsx* o *main.tsx*

- main.tsx
```tsx
<AuthProvider>
  <BrowserRouter>
    <App  />
  </BrowserRouter>
</AuthProvider>
```

- MiComponente.tsx
> **Uso de tabs:** Para usar correctamente las *tabs* se requiere del componente `TabVentana` por cada pestaña que se requiera agregar.

<table>
  <tr>
    <th>Property</th>
    <th>Descripción</th>
  </tr>
  <tr>
    <td>eventKey</td>
    <td>Llave única de cada tab</td>
  </tr>
  <tr>
    <td>title</td>
    <td>Título de la tab</td>
  </tr>
</table>

```tsx
import { VentanaTabs, TabVentana } from 'react-ecosistema-unp/shared';

const MiComponente: React.FC = () => {
  return (
    <VentanaTabs>
    <TabVentana eventKey="idTabUno" title={'Pestaña Uno'}>
      <div>
        Contenido de la pestaña uno...
      </div>
    </TabVentana>
    <TabVentana eventKey="idTabDos" title={'Pestaña Dos'}>
      <div>
        Contenido de la pestaña 2...
      </div>
    </TabVentana>
  </VentanaTabs>
  );
}
```
