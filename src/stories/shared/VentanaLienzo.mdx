# VentanaLienzo
El componente `VentanaLienzo` es un contenedor en blanco. Este incluye el menú lateral del ecosistema.

## Ejemplo de uso

> **Nota:** El componente debe estar envuelto entre el componente `AuthProvider` de **react-ecosistema-unp/utils** y el componente `BrowserRouter` de **react-dom**, es de preferencia hacerlo en el componente *index.tsx* o *main.tsx*

- main.tsx
```tsx
<AuthProvider>
  <BrowserRouter>
    <App  />
  </BrowserRouter>
</AuthProvider>
```

- MiComponente.tsx
```tsx
import { VentanaLienzo } from 'react-ecosistema-unp/shared';

const MiComponente: React.FC = () => {
  return (
    <VentanaLienzo>
    <div>
      Contenido...
    </div>
  </VentanaLienzo>
  );
}
```
