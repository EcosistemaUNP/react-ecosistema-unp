# VentanaLienzo

El componente `VentanaLienzo` es un contenedor flexible que permite incluir cualquier tipo de contenido dentro de él. Aquí puedes ver un ejemplo de cómo usar este componente.

## Ejemplo de uso

Este es un ejemplo básico de cómo puedes usar el componente `VentanaLienzo` con un `children` personalizado:

```
import { VentanaLienzo } from 'react-ecosistema-unp/shared';

export default function App() {
  return (
    <VentanaLienzo>
      <div>Aquí va el contenido del sitio</div>
    </VentanaLienzo>
  );
}
```
